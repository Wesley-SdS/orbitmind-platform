## When to use

Use Web Search quando precisar de informacoes atualizadas que nao estao no contexto do agente. Ideal para pesquisa de mercado, tendencias, dados estatisticos e validacao factual.

## Instructions

1. Formule queries especificas e focadas
2. Use multiplas queries para triangular informacoes
3. Priorize fontes confiáveis (papers, sites oficiais, portais de noticias)
4. Sempre registre a URL da fonte
5. Verifique a data da informacao — descarte dados com mais de 6 meses sem disclaimer

## Best Practices

- Comece com queries broad, depois refine
- Use aspas para buscar frases exatas
- Inclua o ano atual para dados mais recentes
- Busque em portugues E ingles para cobertura maior
- Limite a 5-10 queries por pesquisa para manter foco
