## When to use

Use Document Writer para gerar documentos estruturados: relatorios, briefs, especificacoes tecnicas, posts de blog, newsletters.

## Instructions

### Estrutura

1. Comece com titulo claro e metadata (data, autor, versao)
2. Inclua sumario executivo (2-3 paragrafos)
3. Organize em secoes logicas com headings hierarquicos
4. Use listas e tabelas para dados estruturados
5. Termine com conclusao e proximos passos

### Formatacao

- Markdown como formato padrao
- Headings: H1 para titulo, H2 para secoes, H3 para subsecoes
- Code blocks com syntax highlighting quando relevante
- Links inline para referencias
- Imagens com alt text descritivo

### Qualidade

- Revise ortografia e gramatica
- Verifique consistencia de terminologia
- Garanta que todos os dados citados tem fonte
- Mantenha tom consistente com o brand voice
- Limite paragrafos a 3-4 linhas
